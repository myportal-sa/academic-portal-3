# 🎓 البوابة الأكاديمية المتطورة (Academic Portal)

مشروع ويب متكامل واحترافي لإدارة وعرض بيانات أعضاء هيئة التدريس والمحاضرات، مع تكامل كامل لخرائط Google ونظام صلاحيات متقدم.

## ✨ المميزات الرئيسية
- **واجهة مستخدم عصرية:** تصميم متجاوب (Responsive) باستخدام Bootstrap 5 مع لمسات جمالية واحترافية.
- **نظام صلاحيات (RBAC):** تمييز كامل بين المستخدم العادي (عرض فقط) والمدير (إدارة كاملة).
- **خرائط تفاعلية:** عرض مواقع المكاتب والقاعات الدراسية باستخدام Google Maps API.
- **إدارة البيانات:** إمكانية استيراد البيانات من ملفات CSV بشكل تلقائي.
- **لوحة تحكم المدير:** إحصائيات سريعة، إضافة وحذف الدكاترة، وإدارة المحتوى.
- **قاعدة بيانات متكاملة:** استخدام SQLAlchemy لإدارة الجداول (Doctors, Classes, Users).

## 🛠 التقنيات المستخدمة
- **Backend:** Python Flask
- **Frontend:** HTML5, CSS3, Jinja2 Templates
- **Database:** SQLite (يمكن تحويلها لـ PostgreSQL/Supabase بسهولة)
- **Authentication:** Flask-Login
- **Data Processing:** Pandas (لاستيراد ملفات CSV)
- **Maps:** Google Maps JavaScript API

## 🚀 تعليمات التشغيل

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. تهيئة قاعدة البيانات واستيراد البيانات
```bash
# تهيئة الجداول والمستخدمين الافتراضيين
python3 -c "from app import app, init_db; init_db()"

# استيراد البيانات من ملفات CSV
python3 import_data.py
```

### 3. تشغيل التطبيق
```bash
python3 app.py
```
سيفتح التطبيق على الرابط: `http://127.0.0.1:5000`

## 🔐 بيانات الدخول التجريبية

- **المستخدم العادي (User):**
  - المستخدم: `user`
  - كلمة المرور: `user123`

## 📁 هيكل المشروع
- `app.py`: الملف الرئيسي للتطبيق والمسارات.
- `models.py`: تعريف جداول قاعدة البيانات.
- `config.py`: إعدادات التطبيق.
- `import_data.py`: سكربت استيراد البيانات من CSV.
- `templates/`: قوالب HTML.
- `data/`: ملفات CSV النموذجية.
- `static/`: ملفات CSS و JS والصور.
